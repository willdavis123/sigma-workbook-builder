## Prompt

"Build an executive sales overview dashboard for a 15-minute customer
call. Use whatever sales/revenue data source looks most relevant."

## What good looks like

- Goal line in the plan explicitly notes the exec/short-call audience
- 4-6 KPIs max, each paired with a time dimension
- One trend chart (line, colored by year/period if multi-year data)
- One ranked comparison (horizontal bar, sorted descending)
- A single page, not split across several

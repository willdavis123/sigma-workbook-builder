## Prompt

"Build an ops detail dashboard for the analytics team with region and
date filtering, showing performance by store/account."

## What good looks like

- A detail-oriented table (not just KPIs) since audience = analysts
- Shared date range + region controls driving multiple elements, not
  duplicated per chart
- Table sorted meaningfully (e.g. descending by primary metric)
- Conditional formatting used sparingly, on the column(s) that matter
